1. 系統概觀

1.1 架構

層級技術
後端FastAPI（Python）+ SQLite（monitor.db）
前端單頁應用（SPA），內嵌於 index.html
台股行情永豐金 Shioaji API（即時行情 + 歷史 K 線）
美股行情yfinance（延遲行情）
AI 分析Claude API（訂閱制零費用 或 API 計費）

1.2 畫面配置

  • 頂部標題列：系統名稱、市場切換（TW/US）、風控等級 badge、搜尋框
  • 左側導航列：12 個主頁面按鈕（垂直排列）
  • 主內容區：佔滿右側空間
  • 底部緊急列：EXIT_D 停損警示（紅色，出現時不可關閉）

1.3 市場切換

頂部標題列有 TW 台股 / US 美股 切換按鈕：

  • 切換後，首頁自選股、訊號、持倉皆連動
  • K 線圖搜尋欄位也切換對應市場
  • 某些功能（如籌碼分析）僅限台股

7. 風控中心前往操作 →

導航：點擊左側 🛡 圖示

7.1 MACRO_LOCK 全局鎖定

四個總經條件監控：

  2. DXY 月漲 > 3%

14. 安全與保護機制

EXIT_D 保命機制 — 不可關閉
跌破成本 5% 自動觸發賣出訊號。底部紅色警示條永遠顯示，無法停用。
MACRO_LOCK 全局鎖定
總經異常時自動/手動鎖定所有買入。頂部紅色橫幅動態閃爍提醒。
API Token 保護
所有寫入操作需 token 驗證。敏感資訊（Telegram token、Email 密碼）存於 risk_config，不外洩。

三級風控一覽

等級條件操作限制
NORMAL無觸發全額操作
CAUTION觸發 1 個部位縮至 60%
ALERT觸發 ≥ 2 個部位縮至 30%，禁止買入

附錄：快速操作流程

A. 每日開盤流程

  1. 開啟首頁 → 檢查風控等級
  2. 查看今日訊號（自動 or 手動掃描）
  3. 有 BUY 訊號 → 跳轉 K 線查看 → 快速下單
  4. 檢查持倉 EXIT_D 警示

B. 每日收盤流程

  1. 盤後分析 → 抓取今日籌碼
  2. 資訊中心 → AI建議 → 產生推薦（台股+美股分開）
  3. 評估歷史推薦結果
  4. 交易紀錄 → 補錄今日交易

C. 週末研究流程

  1. 專家委員會 → 新建週度分析
  2. 資訊中心 → 量化工具 → 多因子排名
  3. 系統透視 → 按公式 → 調參數 → 回測驗證
  4. 策略管理 → 根據回測結果調整策略

D. 新增一檔股票

  1. 首頁 → 自選股 → + 新增 → 輸入代碼
  2. 點擊該股 → K 線 → 查看技術面
